# Portfolio Blog

A personal portfolio blog site showcasing my work as a developer: projects, writings, and thoughts on software engineering. The site is designed to be fast, accessible, and easy to maintain.

## Features

- Minimal, clean design focused on content
- Blog posts with tags
- Projects/portfolio listing with details and links
- RSS Feed
- SEO-friendly metadata and shareable links

## Tech Stack

Astro.js (SSG), Tailwind CSS, Static deployment
